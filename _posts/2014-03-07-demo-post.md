# Django

### 템플릿 변수
- 사용 형식 : \{\{ variable \}\}
- 변수명은 일반 프로그래밍 문자, 숫자, 밑줄(_) 사용하여 정의
- 변수 속성 접근도 도트(.)표현식으로 가능

### 템플릿 필터
- 필터는 파이프(\|)문자 사용
- 장고는 약60여가지 필터를 제공하고 있으며, 사용자 정의 필툐도 만들 수 있다.
